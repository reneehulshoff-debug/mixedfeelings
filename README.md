# MIXED FEELINGS - 2.0

**MIXED FEELINGS** is een interactieve, generatieve 3D-visualisatie gebaseerd op het **Delphi-model voor hoogbegaafdheid**. Het project vertaalt abstracte persoonskenmerken naar een dynamische, geometrische vorm.

**Het uitgangspunt** van de visualisatie is een perfecte, gladde bol: De neurotypische "norm". Door interactie met de verschillende parameters kan de gebruiker de vorm boetseren naar een persoonlijke reflectie van eigen zijn, denken en voelen.
